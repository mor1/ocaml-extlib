OCaml Extended standard Library - ExtLib.
-----------------------------------------

Installation :

get and install Ocamake from http://tech.motion-twin.com
run 'make' or 'nmake' on the extlib sources
this will produce a big ext-lib.cma and several cmi.
run 'make opt' to compile the ext-lib.cmxa

Documentation :

run 'make doc' for html documentation
this will create a 'dochtml' directory containing ocamldoc
generated documentation
